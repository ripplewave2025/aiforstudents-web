Lesson: What is a Variable?

In programming, a variable is a container for storing data values. Think of it like a labeled box where you can put information.

// In JavaScript, you declare a variable with 'let' or 'const'
let score = 0; // 'score' is the variable, 0 is its value
score = 10;    // Now the value of 'score' is 10

const name = "Rahul"; // 'const' means the value cannot be changed
